Interactive Kiosk Revolver

by Curtis Bratton

Automatically start Revolver Tabs on browser startup.

Forces the window to full screen whenever a new tab is loaded.

Closes all other open tabs whenever a new tab is loaded.

Reopens the browser window if the user closes it.

Prevents the user from opening the context menu via right clicking.

Grab focus back from other browser windows.


Each URL should be formatted [rotateTime],[rotateTimeWithInteract],[URL]. Example 15,30,https://google.com




Each entry should be a regex. A match will allow the page content to load, otherwise the user will be redirected.

Rotation will pause if the browser has been interacted with within the time specified in the URLs list.
